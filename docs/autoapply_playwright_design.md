# Playwright Automation Design

## Flow

1.  Launch browser with stealth settings\
2.  Navigate to job platform\
3.  Search for matching roles\
4.  Open Easy Apply form\
5.  Upload tailored resume\
6.  Fill required fields\
7.  Submit application\
8.  Log success/failure

## Anti-Bot Measures

-   Randomized delays
-   Rotating user agents
-   IP rotation via proxy
